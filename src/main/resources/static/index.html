Endpoints da API

GET
/api/usuarios
Lista todos os usuários cadastrados
GET
/api/usuarios/{id}
Busca um usuário específico pelo ID
POST
/api/usuarios
Cria um novo usuário
PUT
/api/usuarios/{id}
Atualiza os dados de um usuário
DELETE
/api/usuarios/{id}
Remove um usuário do sistema
